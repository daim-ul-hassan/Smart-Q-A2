🚀 DEPLOY IN 5 STEPS:
---------------------

1. GO TO STREAMLIT CLOUD
   Visit: https://streamlit.io/cloud
   Sign in with your GitHub account

2. CREATE NEW APP
   Click "New app"
   Repository: daim-ul-hassan/Smart-Q-A2
   Branch: main
   Main file: app.py

3. ADD YOUR API KEY (IMPORTANT!)
   Click "Advanced settings"
   In "Secrets", add:
   
   [google]
   api_key = "YOUR_GOOGLE_API_KEY_HERE"
   
   Get your key from: https://aistudio.google.com/apikey

4. DEPLOY
   Click "Deploy!"
   Wait 2-5 minutes

5. ACCESS YOUR APP
   Your app will be live at:
   https://your-app-name.streamlit.app

⚠️  IMPORTANT LIMITATION:
-------------------------
Streamlit Cloud doesn't have persistent storage!

This means:
❌ Uploaded documents disappear after each session
❌ Vector store (chroma_db/) won't persist

SOLUTIONS:
Option 1: Pre-build vector store locally and commit to Git
Option 2: Use cloud storage (Google Drive, AWS S3, etc.)
Option 3: Keep Streamlit Cloud for demo, run locally for full use
